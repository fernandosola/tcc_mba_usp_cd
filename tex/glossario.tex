
\newword{Pipeline}{Na ciência de dados \textit{pipeline} refere-se a uma série de etapas necessárias para o completo tratamento de dados}

\newword{Feature}{Uma variável ou característica que faz parte de uma observação ou registro}

\newword{Outlier}{Em estatística, outlier, valor aberrante ou valor atípico, é uma observação que apresenta um grande afastamento das demais da série, ou que é inconsistente}
